PWA E APLICAÇÕES MODERNAS

Definição

PWA ou aplicativos web progressivos são simplesmente a websites que você pode instalar da mesma forma que um aplicativo da Playstore. Eles trazem os bônus dos aplicativos (conveniência e usabilidade off-line) e o melhor dos websites (Atualizações rápidas e peso otimizado).
Aqui vão alguns dos benefícios dos PWA:
-Velocidade e fácil instalação
-Requisitos de armazenamento mínimos
-Performance confiável off-line
-Compatibilidade entre dispositivos
-Updates automáticos e notificações

SERVICE WORKERS

São scripts que rodam atrás no backend, separados da sua página web. Eles agem como uma ponte entre a aplicação web e a internet. Seu principal papel é garantir que seu app fale com a internet.
Eles podem salvar partes importantes no dispositivo do usuário, o que significa que seu aplicativo pode funcionar mesmo sem internet. Também podem atualizar em silêncio o conteúdo da aplicação e enviar notificações, assim como um aplicativo nativo.

MANIFEST

Conhecido como app manifest, são um arquivo JSON. São essenciais pois dizem ao dispositivo do usuário como o aplicativo deve parecer e se comportar.
É como a identificação do seu aplicativo- ele inclui o nome do aplicativo, seus icones, como a primeira página deve abrir e como sera demonstrada. Esse arquivo faz seu web app parecer mais um aplicativo regular.

CACHE

É vital para o funcionamento de aplicações off-line, é como a memória do seu aplicativo. Ele guarda partes importantes para serem carregadas rapidamente depois, mesmo com a ausência de internet. É crucial para o app funcionar off-line.
Dentro das PWA's há três formas de lidar com o cache: sendo a primeira cache-first (o app verifica o cache antes da internet), network-first (o oposto), e stale-while-revalidate (uma mistura de ambos). A escolha depende do funcionamento do seu app

Exemplo de código

1 - Configure o Manifest { "name": "Ophy's To-Do List", "short_name": "OphyToDo", "icons": [ { "src": "favicon.ico", "sizes": "64x64 32x32 24x24 16x16", "type": "image/x-icon" } ], "start_url": "/", "display": "standalone", "theme_color": "#000000", "background_color": "#ffffff" }
2 - Configure o script self.addEventListener('install', function(event) { // Perform install steps var CACHE_NAME = 'ophy-todo-cache-v1'; var urlsToCache = [ '/', '/styles.css', '/app.js' ]; event.waitUntil( caches.open(CACHE_NAME) .then(function(cache) { console.log('Opened cache'); return cache.addAll(urlsToCache); }) ); });

USOS REAIS

exemplos de empresas que utilizam

Khan-Academy

Um site educacional bem conhecido que usa o PWA que permite os estudantes acessem e instalarem as lições e cursos sem disponibilidade de internet. Particularmente útil em locais sem acesso

Starbucks

Tem um PWA que permite os clientes verem o cardápio e fazerem pedidos mesmo que não tenham acesso a internet. Isso torna a experiência e vendas para os consumidores ainda melhor.

Trivago

A empresa conhecida pelo seu serviço como um motor de buscas focado em hotéis, desenvolveu um PWA que permite usuários terem acesso ao catálogo mesmo sem conexão à internet. Isso contribuiu para o aumento de conversão e engajamento do usuário.
